symbols 폴더
======================
Footprint(Symbol)을 저장하는 경로
* 확장자
	* .DRA : drawing Data (Symbol을 생성 시 사용하는 데이터, PAD와 symbol 정보를 가지고 있다.)
	* .PSM : Package Symbol (1개 이상의 전기적으로 연결할 핀을 가지고 있다.)
	* .BSM : Mechanical Symbol (기구홀이나 아웃라인 데이터)
	* .OSM : Format Symbol (서식, 기호, 핀, 선, 메모 등)
	* .SSM : Shape Symbol (특이한 모양의 이형 패드 그림)
	* .FSM : Flash Symbol (음각 전원층[Plane]의 단열판 정보)
